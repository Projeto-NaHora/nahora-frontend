# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started
### Antes de tudo, no terminal, após clonar o projeto, tenha certeza que está dentro do diretório /nahora
  ```bash
  cd nahora
  ```

1. Setup android emulator with android studio
  - https://docs.expo.dev/get-started/set-up-your-environment/?platform=android&device=simulated&mode=development-build
  - Sigam todos os passos até chegar em "Create a development build", a partir dali, ignorem o resto e rodem o emulador pelo menos uma vez, então sigam em frente com os passos deste documento.
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo prebuild
   npx expo run:android
   ```

### A partir daqui, pode-se iniciar o desenvolvimento a partir do template presente no projeto, ainda pretendo fazer algumas alterações no conteúdo e estrutura do template, mas vou atualizando vcs no zap

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).
